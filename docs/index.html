🌐 Búsqueda Global (Investigación)

¿Para qué sirve la Búsqueda Global?

Esta función te permite investigar buscando en TODOS los juzgados y salas al mismo tiempo.

Casos de uso:

  • Buscar todas las causas de una persona
  • Verificar si existe un expediente (sin saber el juzgado)
  • Investigar antecedentes legales

Nota: Abre múltiples ventanas. El navegador puede bloquear popups - permite las ventanas emergentes del sitio TSJ.

Busca un número de expediente o nombre en TODOS los juzgados y salas a la vez.

🤖 Analizar Acuerdo con IA

¿Cómo usar el análisis con IA?

Paso 1: Configura tu API Key de Groq en Configuración (es gratis)

Paso 2: Busca tu expediente y copia el texto del acuerdo

Paso 3: Pega el texto aquí y haz clic en "Analizar"

La IA extraerá:

  • Fechas de audiencias
  • Vencimientos de plazos
  • Puntos importantes del acuerdo

Puedes guardar automáticamente como eventos y notas.

Pega el texto de un acuerdo judicial (TSJ o PJF) y la IA extraerá automáticamente el número de expediente, fechas, audiencias y puntos importantes.

🔍 Consulta de Expedientes Federales

¿Cómo funciona?

  • Selecciona un circuito para filtrar los órganos jurisdiccionales
  • Selecciona un órgano para ver los tipos de asunto disponibles
  • Ingresa el número de expediente (formato: 67/2021)
  • La consulta abre el expediente en el portal DGEJ del PJF
Ingresa el ID numérico del tipo de asunto del SISE
Formato: número/año
Solo aplica para Tribunales Laborales

📁 Mis Expedientes Federales

¿Cómo usar la carga masiva PJF?

Paso 1: Descarga el template CSV

Paso 2: Ábrelo con Excel o Google Sheets

Paso 3: Completa las columnas:

  • expediente: Número (ej: 67/2021)
  • organo: Nombre EXACTO del órgano
  • organismo_id: ID numérico (opcional)
  • tipo_asunto_id: ID del tipo (opcional)
  • comentario: Notas opcionales

Paso 4: Guarda como CSV e importa

Con organismo_id y tipo_asunto_id el botón "🔍 Buscar" abrirá el portal PJF directamente.

📧 Email (EmailJS)

¿Qué es EmailJS?

EmailJS te permite recibir recordatorios por email sin necesidad de un servidor.

¿Para qué sirve?

  • Recibir alertas de eventos del calendario
  • Recordatorios de vencimientos
  • Notificaciones importantes

Configuración (5 min):

  • Crea cuenta en emailjs.com
  • Conecta tu email (Gmail, Outlook, etc.)
  • Crea un "Email Service" y copia el ID
  • Crea un "Template" y copia el ID
  • Copia tu "Public Key" de Account

Plan gratis: 200 emails/mes
